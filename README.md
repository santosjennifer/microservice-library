## Library API
Projeto para cadastro e controle de aluguel de livros. 

Envia um alerta customizado por e-mail caso o cliente atrase a devolução do livro.

### Tecnologia:
* Java 21
* Maven
* Spring Boot 3.2.4
* PostgreSQL
